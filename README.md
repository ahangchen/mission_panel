# Mission Panel

家庭自动化系统的Web控制面板

## 功能特性

- 📊 **任务监控**: 查看过去一周的任务状态（完成/进行中/失败）
- 📁 **文件浏览器**: 浏览和查看coding目录下的项目文件
- 📈 **使用统计**: Skill使用次数排名和模型消耗统计

## 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **后端**: Python + FastAPI + SQLite
- **部署**: Docker + Nginx
